# jupyter-cc-magic

## Tech Stack

- Python 3.13
- Package manager: uv
- Linting/Formatting: ruff
- Testing: pytest


## Project Structure

- `src/jupyter_cc_magic/` - Main package code
- `tests/` - Test files
- `docs/` - Documentation
- `docs/plans/` - Implementation plans


## Configuration

All tool configuration is in `pyproject.toml`. Check there for:

- Ruff lint rules and settings
- Pytest configuration
- Dependencies

## Documentation

- `docs/product-design.md` - Product requirements and design
- `docs/architecture.md` - Technical design decisions
